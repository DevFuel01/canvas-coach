### AI Visual Art Coach for Beginners

Canvas Coach is an AI-assisted creativity web application that helps **absolute beginners learn visual art through step-by-step coaching**.  
Instead of creating artwork for users, the AI acts as a **mentor and guide**, explaining artistic principles, giving constructive feedback, and suggesting clear next steps while the user creates their own original art.

This project was built for the **InnovArt International Hackathon**, focusing on ethical, human-centered AI in creative education.
